# While You Were Away

Scratchpad for quick notes and pending tasks.

---

## Purpose

This is the `claude-env` repository — a standalone development environment repo.

**Feature requests and project-specific issues:** Add to individual project repos (stock-analyzer, road-trip, etc.) instead.

---

## Current Status

`claude-env` is a reusable development environment containing:
- Git hooks for code safety and compliance
- Python helper scripts (testing, deployment, validation)
- Bash and PowerShell scripts for automation
- WSL2 sandbox setup
- Integration with app repositories

**Not included:** App-specific code (Stock Analyzer, Road Trip, etc.)
